# Data Directory

This directory should contain the following data files (not included in repo due to size):

## Required Files:
- `veteran_rag_corpus.csv` - Processed corpus (52MB)
- `female_veterans_clean.csv` - Female veteran data (2.7MB)
- `male_veterans_clean.csv` - Male veteran data (11MB)
- `Sample_Characteristics_20251123.csv` - VA sample data (1MB)
- `Abbreviations_20251123.csv` - Medical abbreviations (6.7KB)
- `LLCP2024.XPT` - Raw BRFSS data (1GB)
- `pharma-nonpharma-ptsd-2023-update-app-g-2/` - Treatment research files

## Where to Get Data:
See [DATA_SOURCES.md](../DATA_SOURCES.md) for download links.
